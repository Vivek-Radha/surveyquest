# NextGen Survey System

A comprehensive survey system comprising a frontend and backend architecture.

## Features
- Create, manage, and share surveys
- Analyze survey responses with an integrated dashboard
- Modern, dynamic user interface (React/Vite)
- Robust backend API (Node.js/Express)

## Starting the Application

From this root directory, you can install all dependencies and start both the frontend and backend servers concurrently:

```bash
# Install dependencies for root, frontend, and backend
npm run install-all

# Start both frontend and backend
npm start
```
